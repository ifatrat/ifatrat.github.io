MiniGUI3.0移植到Hisi3521A

发表于 2016-07-19

1、编译zlib [这个必须优先于libpng库编译]

  • 解压: tar -xvf zlib-1.2.2.tar.gz
  • 进入解压目录: cd zlib-1.2.2
  • 编辑Makefile:
    • 修改19行: CC=arm-hisiv300-linux-gcc
    • 修改36行: AR=arm-hisiv300-linux-gcc rc
    • 修改37行: RANLIB=arm-hisiv300-linux-ranlib
  • 编译：make
  • 拷贝:
    • cp zlib.h zconf.h ../../MiniGUI/include/
    • cp libz.a ../../MiniGUI/lib/

2、编译libpng

  • 解压：tar -xvf libpng-1.2.37.tar.gz
  • 配置： LDFLAGS=”-L../../MiniGUI/lib/“ CPPFLAGS=”-I../../MiniGUI/include” ./configure –host=arm-hisiv300-linux
  • 编译：make
  • 拷贝：
    • cp png.h pngconf.h ../../MiniGUI/include
    • cp -a .libs/libpng.so* ../../MiniGUI/lib/

3、编译jpeg:

  • 解压：tar -xvf jpegsrc.v7.tar.gz
  • 配置：./configure –host=arm-hisiv300-linux
  • 编译：make
  • 拷贝:
    • cp jconfig.h jerror.h jmorecfg.h jpeglib.h ../../MiniGUI/include/
    • cp -a .libs/libjpeg.so* ../../MiniGUI/lib/

4、安装minigui资源文件:

  • 解压：tar -xvf minigui-res-be-3.0.12.tar.gz
  • 拷贝：cp bmp cursor font icon ../../MiniGUI/res/ -fr

5、编译libminigui-gpl:

  • 解压：tar -xvf libminigui-gpl-3.0.12.tar.gz
  • 打补丁：patch src/newgal/fbcon/fbvideo.c fbvideo.p
  • 配置：CFLAGS=”-I/home/zhenghao/Work/Projects/Tes/Source/Device/OSD/MiniGUI/include” LDFLAGS=”-L ../../MiniGUI/lib” ./configure –host=arm-hisiv300-linux –disable-screensaver –disable-splash –enable-videoqvfb=no –enable-rtosxvfb=no –enable-pcxvfb=no
  • 编译：make
  • 拷贝：
    • 先安装到一个临时目录：make install DESTDIR=/tmp/xx
    • 拷贝
      • cp -a /tmp/xx/usr/local/lib/libminigui_ths* ../../MiniGUI/lib/
      • cp -fr /tmp/xx/usr/local/include/minigui ../../MiniGUI/include/

6、编译sample:

  • 解压:
  • 配置：CPPFLAGS=”-I/home/zhenghao/Work/Projects/Tes/Source/Device/OSD/MiniGUI/include” CFLAGS=”-I/home/zhenghao/Work/Projects/Tes/Source/Device/OSD/MiniGUI/include” LDFLAGS=”-L/home/zhenghao/Work/Projects/Tes/Source/Device/OSD/MiniGUI/lib” MINIGUI_CFLAGS=”-I/home/zhenghao/Work/Projects/Tes/Source/Device/OSD/MiniGUI/include” MINIGUI_LIBS=”-lminigui_ths -lpng -lm -ljpeg -lpthread” ./configure –host=arm-hisiv300-linux
  • 编译：make

7、运行程序前的相关配置

  • 环境变量导入:
    • export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:../../lib
      export MG_RES_PATH=../../res
      export MG_CFG_PATH=../../
  • MINIGUI.cfg
    • gal_engine=fbcon
    • [fbcon]defaultmode=1280x720-16bpp
